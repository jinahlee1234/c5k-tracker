Couch to 5K for AI — Challenge Tracker
A shared progress tracker for groups doing the Couch to 5K for AI challenge together.
Live tracker → jinahlee1234.github.io/c5k-tracker

What is this?
Couch to 5K for AI is a 30-day challenge by Hilary Gridley — 10 minutes a day for 4 weeks to go from chatting with AI to actually building with it.
This tracker was built so a group can do the challenge together and see each other's progress in real time.
Features

Join with your name — no login required
Check off each day as you complete it
See everyone's progress at a glance
Tracks when you joined and when you finished
Filters by week
Data synced via Google Sheets

Built with

Vanilla HTML/CSS/JS
Google Apps Script (backend API)
Google Sheets (database)
GitHub Pages (hosting)
